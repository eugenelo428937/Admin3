# API Contract: Store Admin Endpoints

**Branch**: `20260216-admin-panel-api` | **Base URL**: `/api/store/`

Existing read-only endpoints upgraded to full CRUD:
- **Read (GET)**: `AllowAny` (unchanged — backward compatible)
- **Write (POST/PUT/PATCH/DELETE)**: `IsSuperUser` (new)

---

## Store Products — `/api/store/products/`

### `GET /api/store/products/` (EXISTING — NO CHANGE)

Returns unified product+bundle list. Response format unchanged for backward compatibility.

### `GET /api/store/products/{id}/` (EXISTING — NO CHANGE)

Returns single product detail. Response format unchanged.

### `GET /api/store/products/{id}/prices/` (EXISTING — NO CHANGE)

Returns prices for a product. Response format unchanged.

### `POST /api/store/products/` (NEW)

Create a store product.

**Request**:
```json
{
  "exam_session_subject": 1,
  "product_product_variation": 1,
  "is_active": true
}
```

**Response 201**:
```json
{
  "id": 1,
  "product_code": "CM2/PC/2025-04",
  "exam_session_subject": 1,
  "product_product_variation": 1,
  "is_active": true,
  "created_at": "2026-02-16T10:00:00Z",
  "updated_at": "2026-02-16T10:00:00Z"
}
```

Note: `product_code` is auto-generated by the model's `save()` method.

**Response 400**: Unique constraint violation or invalid FK
**Response 403**: Not superuser

### `PUT /api/store/products/{id}/` (NEW)

**Request**: Same as POST
**Response 200**: Updated object

### `DELETE /api/store/products/{id}/` (NEW)

**Response 204**: Deleted
**Response 400**: Has dependent records (cart items, bundle products, prices)
**Response 403**: Not superuser

## Store Bundle Products — `/api/store/bundle-products/` (NEW ENDPOINT)

### `GET /api/store/bundle-products/`

List all store bundle-product associations.

**Response 200**:
```json
[
  {
    "id": 1,
    "bundle": 1,
    "product": 1,
    "product_code": "CM2/PC/2025-04",
    "default_price_type": "standard",
    "quantity": 1,
    "sort_order": 0,
    "is_active": true
  }
]
```

### `POST /api/store/bundle-products/`

**Request**:
```json
{
  "bundle": 1,
  "product": 1,
  "default_price_type": "standard",
  "quantity": 1,
  "sort_order": 0,
  "is_active": true
}
```

**Response 201/400/403**: Standard patterns

### `GET/PUT/DELETE /api/store/bundle-products/{id}/`

Standard CRUD.
