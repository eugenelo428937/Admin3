Testing Checkout Expired Deadlines

Test Scenario: User has a marking product (CM1 Series X Assignments Marking) in their cart with expired deadlines.

Expected Behavior

When the user reaches the checkout page, they should see a warning message in step 1 (Cart Review) that shows:

⚠️ Marking Deadline Warning

5/5 deadlines for CM1 Series X Assignments (Marking) has expired.

If you need marking for this study session then please purchase Marking Vouchers instead.

Expired papers: X1, X2, X3, X4, X5

Implementation Details

1. Rules Engine Evaluation

Entry Point: checkout_start Rule: "Expired Marking Deadline Detection" (ID: 13) Template: "expired_marking_deadline_warning" (ID: 9) Custom Function: check_expired_marking_deadlines()

2. Frontend Integration

Component: CheckoutSteps.js Location: Step 1 (Cart Review) State: checkoutMessages, checkoutMessagesLoading Evaluation: useEffect on cartItems change Display: JsonContentRenderer for JSON content

3. Data Flow

1. User navigates to checkout page 2. CheckoutSteps component mounts 3. useEffect triggers rulesEngineService.evaluateRulesAtEntryPoint('checkout_start') 4. Backend evaluates custom function check_expired_marking_deadlines() 5. Function finds expired CM1 marking papers (X1-X5 all expired) 6. Returns structured message with template variables 7. Frontend displays warning card in Cart Review step

Test Steps

  1. Add CM1 Series X Assignments (Marking) to cart
  2. Navigate to checkout page
  3. Verify warning appears in step 1 before cart items
  4. Check that warning shows correct expired count (5/5)
  5. Verify expired papers list (X1, X2, X3, X4, X5)
  6. Confirm styling matches warning theme (yellow/orange)

Backend Rule Details

Rule Configuration: - Entry Point: checkout_start - Condition: Custom function check_expired_marking_deadlines returns has_expired_deadlines=true - Action: Display message using template with variables: * expired_count: 5 * paper_count: 5 * subject: CM1 * product_name: Series X Assignments (Marking) * expired_papers: X1, X2, X3, X4, X5
